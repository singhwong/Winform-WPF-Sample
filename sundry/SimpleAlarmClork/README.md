# SimpleAlarmClork

This is a simple timer alarm clork

winform 版的简单闹钟

铃声路径需要改为自己的本地路径，格式为.wav

目前有“起床”和“睡觉”两个提示框可选

增加了’自定义提醒框内容’功能

三分钟后铃声会停止

做了些显示优化

我使用的音频文件格式转换工具 http://www.pcfreetime.com/
